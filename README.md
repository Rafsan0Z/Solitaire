# Solitaire
A web app version of the game Solitaire

Features to be implemented:

1. Ensure all generated games will be solvable
2. Create an interactive prototype that allowed for clickable cards
3. Make the cards appear as they should and simulate a game
4. Implement a fully functional game that can be completed from start to finish
5. Implement an undo feature
6. Implement a redo feature
7. Put a few new themes for the cards and background
8. Differentiate between easy, medium and hard games
9. Implement a 3 tier difficulity system that is functional
10. Implement a scoring system
11. Implement a leaderboard system
